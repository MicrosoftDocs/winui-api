# Microsoft.UI.Xaml.RectHelper.GetLeft

## -description

Gets a "Left" value for the specified [Rect](/uwp/api/windows.foundation.rect). So long as **Width** is positive, "Left" is evaluated as **X**. C# and Microsoft Visual Basic code should use [Rect.Left](/dotnet/api/windows.foundation.rect.left?view=dotnet-uwp-10.0&preserve-view=true) instead.

## -parameters

### -param target

The [Rect](/uwp/api/windows.foundation.rect) to evaluate.

## -returns

The evaluated "Left" value.
