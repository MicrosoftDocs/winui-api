# Microsoft.UI.Xaml.Documents.Typography.SetCapitals

## -description
Sets the value of the [Capitals](/windows/winui/api/microsoft.ui.xaml.documents.typography#xaml-attached-properties) attached property for a specified dependency object.

## -parameters
### -param element
The dependency object for which to set the value of the [Capitals](/windows/winui/api/microsoft.ui.xaml.documents.typography#xaml-attached-properties) property.

### -param value
The new value to set the property to.
