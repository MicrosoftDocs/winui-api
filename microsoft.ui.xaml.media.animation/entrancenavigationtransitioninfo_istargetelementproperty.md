# Microsoft.UI.Xaml.Media.Animation.EntranceNavigationTransitionInfo.IsTargetElementProperty

## -description
Identifies the [EntranceNavigationTransitionInfo.IsTargetElement](/windows/winui/api/microsoft.ui.xaml.media.animation.entrancenavigationtransitioninfo#xaml-attached-properties) XAML attached property.

## -property-value
The identifier for the [EntranceNavigationTransitionInfo.IsTargetElement](/windows/winui/api/microsoft.ui.xaml.media.animation.entrancenavigationtransitioninfo#xaml-attached-properties) XAML attached property.

## -remarks
This property is only an identifier for the property system, and isn't used in most app scenarios. In most cases you set the [EntranceNavigationTransitionInfo.IsTargetElement](/windows/winui/api/microsoft.ui.xaml.media.animation.entrancenavigationtransitioninfo#xaml-attached-properties) XAML attached property in XAML and won't need this identifier.
