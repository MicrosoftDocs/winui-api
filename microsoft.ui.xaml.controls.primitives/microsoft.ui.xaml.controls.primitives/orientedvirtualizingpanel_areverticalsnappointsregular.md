# Microsoft.UI.Xaml.Controls.Primitives.OrientedVirtualizingPanel.AreVerticalSnapPointsRegular

## -description
Gets a value that indicates whether the vertical snap points for the [OrientedVirtualizingPanel](orientedvirtualizingpanel.md) are equidistant from each other.

**This documentation applies to WinUI 2 for UWP** (for [WinUI](/windows/apps/winui/winui3/) in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the vertical snap points for the [OrientedVirtualizingPanel](orientedvirtualizingpanel.md) are equidistant from each other; otherwise, **false**.
