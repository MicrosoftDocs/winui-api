# Microsoft.UI.Xaml.Media.ScaleTransform.ScaleXProperty

## -description
Identifies the [ScaleX](scaletransform_scalex.md) dependency property.

## -property-value
The identifier for the [ScaleX](scaletransform_scalex.md) dependency property.
