# Microsoft.UI.Xaml.Media.MatrixHelper.GetIsIdentity

## -description
Returns whether the provided [Matrix](matrix.md) represents an **Identity** transform. C**This documentation applies to WinUI 2 for UWP** for [WinUI]/windows/apps/winui/winui3/ in the [Windows App SDK]/windows/apps/windows-app-sdk/, see the **[Windows App SDK namespaces]/windows/windows-app-sdk/api/winrt/**.

# and Microsoft Visual Basic code should use [IsIdentity](matrix_isidentity.md) instead.

## -parameters
### -param target
The target [Matrix](matrix.md) to evaluate.

## -returns
**true** if the [Matrix](matrix.md) represents an **Identity** transform; otherwise, **false**.
