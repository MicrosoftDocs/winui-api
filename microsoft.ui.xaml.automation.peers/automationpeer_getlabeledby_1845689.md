# Microsoft.UI.Xaml.Automation.Peers.AutomationPeer.GetLabeledBy

## -description
Gets the [AutomationPeer](automationpeer.md) for the [UIElement](../microsoft.ui.xaml/uielement.md) that is targeted to the element.

**This documentation applies to WinUI 2 for UWP** for [WinUI]/windows/apps/winui/winui3/ in the [Windows App SDK]/windows/apps/windows-app-sdk/, see the **[Windows App SDK namespaces]/windows/windows-app-sdk/api/winrt/**.

## -returns
The [AutomationPeer](automationpeer.md) for the element that is targeted by the [UIElement](../microsoft.ui.xaml/uielement.md).
