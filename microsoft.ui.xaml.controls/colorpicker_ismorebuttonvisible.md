# Microsoft.UI.Xaml.Controls.ColorPicker.IsMoreButtonVisible


## -description

Gets or sets a value that indicates whether the 'more' button is shown.


## -property-value

**true** if the 'more' button is shown; otherwise, **false**. The default is **false**.
