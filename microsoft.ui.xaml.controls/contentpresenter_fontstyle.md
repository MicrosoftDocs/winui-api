# Microsoft.UI.Xaml.Controls.ContentPresenter.FontStyle

## -description
Gets or sets the font style for the presented content.

**This documentation applies to WinUI 2 for UWP** (for [WinUI](/windows/apps/winui/winui3/) in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

The requested font style, which is a [FontStyle](/uwp/api/windows.ui.text.fontstyle) constant name. The default is Normal.
