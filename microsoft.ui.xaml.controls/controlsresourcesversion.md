# Microsoft.UI.Xaml.Controls.ControlsResourcesVersion

<!--

## -description

Defines constants for specifying the version of WinUI control styles to use.

## -enum-fields

### -field Version1: 1

Legacy style. This is the default.

### -field Version2: 2

Latest style.
